# 🎬 Cine Search – Movie Search Web App

## 📌 Project Overview

Cine Search is a Netflix-inspired movie search web application that allows users to explore trending movies using real-time data from a public API. Users can search for movies, sort results, and save their favorite content.

---

## 🚀 Features

### 🔍 Search

* Search for movies using keywords
* Dynamic results fetched from API

### 🎬 Movie Display

* Poster image
* Movie title
* Rating

### 🔽 Sorting

* Sort by rating (High → Low)
* Sort by rating (Low → High)

### ❤️ Favorites

* Save favorite movies using localStorage
* Persistent data after page reload

### 🌙 Dark Mode

* Toggle between light and dark theme

---
